## 工具介绍  
  
这是一个基于大语言模型（LLM）的多智能体全流程漏洞挖掘项目。它支持搭建PHP、Java、Python、Go、Node.js等多种语言项目的环境，并涵盖漏洞分析、漏洞验证及报告生成等环节。通过支持多智能体并发，能够高效完成漏洞挖掘任务。  
  
这是一个AI驱动的代码安全审计平台前端。用户可上传源代码压缩包，后端将启动Docker容器进行漏洞分析，分析结果会通过WebSocket实时回传。  
## 功能特性  
- 📦 **源码上传**  
 - 支持上传 .zip  
 或 .tar.gz  
 格式文件，最大体积为500MB。  
  
- 🔍 **漏洞扫描**  
 - 采用AI技术进行自动化的代码安全分析。  
  
- 📡 **实时反馈**  
 - 通过WebSocket实时推送扫描进度和结果。  
  
- 🐳 **容器管理**  
 - 可查看审计容器的运行状态。  
  
- 📊 **报告生成**  
 - 自动生成Markdown格式的审计报告。  
  
- 🌙 **赛博主题**  
 - 采用深色科技风格的界面设计。
