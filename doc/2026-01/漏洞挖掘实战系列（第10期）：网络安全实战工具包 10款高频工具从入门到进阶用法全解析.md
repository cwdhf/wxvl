## 一、Web渗透工具（核心4款）  
### 1. Burp Suite：Web渗透的“瑞士军刀”  
  
**核心用途**  
：抓包、改包、扫描、爆破、自动化测试。  
  
**入门必会：**  
- Proxy抓包  
  
- Repeater重放  
  
- Intruder爆破  
  
- Scanner主动/被动扫描  
  
**进阶技巧（高手常用）：**  
- 自定义Payload：用“Cluster bomb”模式做SQL盲注爆破  
  
- 宏（Macro）自动登录：解决需要登录的漏洞测试  
  
- 插件：  
  
- Logger++ 抓包日志增强  
  
- Retire.js 检测前端库漏洞  
  
- J2EEScan 检测Java漏洞  
  
**实战场景：**  
  
用Burp宏自动登录 → 扫描后台 → 发现越权漏洞。

## 二、Pwn & 逆向工具（核心3款）  
### 5. IDA Pro：逆向分析神器  
  
**核心用途**  
：反编译二进制、找漏洞、看逻辑。  
  
**入门必会：**  
- 看伪代码  
  
- 找字符串  
  
- 找交叉引用（Xref）  
  
**进阶技巧：**  
- 用IDC脚本自动分析  
  
- 用Hex-Rays反编译插件看更清晰的伪代码  
  
- 用F5修复伪代码（变量重命名、结构体定义）  
  
**实战场景：**  
  
找到gets(buf)  
 → 确定栈溢出 → 算偏移 → 写exp。  
### 6. GDB + pwndbg：动态调试神器  
  
**核心用途**  
：调试Pwn程序、算偏移、看寄存器。  
  
**入门必会：**  
- r  
 运行  
  
- n  
 下一步  
  
- x/20xw $esp  
 看栈  
  
- pattern create / pattern offset  
 算偏移  
  
**进阶技巧：**  
- 断点：b *0x401234  
  
- 查看寄存器：info registers  
  
- 查看栈帧：bt  
  
**实战场景：**  
  
算栈溢出偏移 → 找canary → 构造ROP链。  
### 7. pwntools：写exp的万能库  
  
**核心用途**  
：远程连接、打包payload、自动化攻击。  
  
**入门必会：**  
  
![](https://mmbiz.qpic.cn/sz_mmbiz_png/iaLzURuoralb27icrPbNPgXbDO77Fvq6iaFrc199dFKn1cqLROmCOmm2Pf8qprzmdMJWPelZuK9RE1p0l9hStLSbg/640?wx_fmt=png&from=appmsg "")  
  
**进阶技巧：**  
- 自动打包：flat()  
  
- ELF文件解析：elf.sym["main"]  
  
- 上下文设置：context.arch = "amd64"  
  
**实战场景：**  
  
写栈溢出、UAF、ROP链的exp。  
## 三、应急响应工具（核心3款）  
### 8. Wireshark：流量分析神器  
  
**核心用途**  
：抓包、分析攻击流量、找漏洞利用痕迹。  
  
**入门必会：**  
- 过滤HTTP：http  
  
- 过滤TCP：tcp.port == 8080  
  
- 追踪流：右键 → Follow → TCP Stream  
  
**进阶技巧：**  
- 过滤异常流量：tcp.flags.reset == 1  
  
- 导出对象：File → Export Objects → HTTP  
  
- 分析DNS隧道：dns.qry.name contains "c2."  
  
**实战场景：**  
  
分析勒索病毒流量 → 找到C2服务器IP。  
### 9. Volatility：内存取证神器  
  
**核心用途**  
：分析内存镜像、找进程、找后门、找密码。  
  
**入门必会：**  
  
![](https://mmbiz.qpic.cn/sz_mmbiz_png/iaLzURuoralb27icrPbNPgXbDO77Fvq6iaFUES9AFIEYbRIssLpbpIfibsrFFQQhBqGSq6pnk5jnNBXXaZzHITTumw/640?wx_fmt=png&from=appmsg "")  
  
**进阶技巧：**  
- 找隐藏进程：psscan  
  
- 找网络连接：connscan  
  
- 提取注册表：hivelist  
  
- 找密码：hashdump  
  
**实战场景：**  
  
分析被入侵服务器内存 → 找到木马进程和攻击者IP。  
### 10. Ghidra：免费逆向神器（NSA出品）  
  
**核心用途**  
：逆向二进制、分析恶意代码、找漏洞。  
  
**入门必会：**  
- 导入文件  
  
- 看伪代码  
  
- 找字符串  
  
**进阶技巧：**  
- 定义结构体  
  
- 批量重命名变量  
  
- 用脚本自动分析  
  
**实战场景：**  
  
分析恶意软件 → 找到C2域名 → 溯源攻击者。
